Operating model

Workflow work breaks when the operator cannot see where a contact came from or why they received a message. This page frames the skill as a routing discipline: agents inspect context, name the hidden assumptions, and package safe next steps.

Best forTeams with journeys spread across docs, builders, and analytics
Primary artifactTrigger map with gaps, collisions, and next actions
Stop conditionAny edit that changes production audience flow

Where it helps

Three concrete jobs this skill is built to handle.

Lifecycle map audit

Trace welcome, activation, expansion, and winback paths before a teammate changes logic.

Trigger logic review

Ask the agent to compare event names, filters, delays, and exit rules.

Sequence handoff plan

Return a clean build brief with owner, inputs, risks, and approval state.
